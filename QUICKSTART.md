# 🚀 QUICK START (5 minutes)

## 1️⃣ Get API Key (2 min)
- Visit: https://ai.google.dev/
- Click **"Get API Key"**
- Click **"Create API key in new project"**
- Copy your API key

## 2️⃣ Create .env File (1 min)
1. In the project folder, create a file named `.env`
2. Add this:
```
GEMINI_API_KEY=your_api_key_here
```
Replace `your_api_key_here` with your actual key

## 3️⃣ Install & Run (2 min)
Open PowerShell in project folder and run:
```powershell
npm install
```

Wait for completion, then run:
```powershell
node server.js
```

You'll see ✅ `🚀 WCE Chatbot server running on http://localhost:3000`

## 4️⃣ Test the Chatbot
1. Open `wce_homepage(2).html` in browser
2. Click the **🤖** button (bottom-right)
3. Type "Hello" and chat!

---

## ⚠️ Common Mistakes
❌ Don't close PowerShell (server must stay running)
❌ Don't forget the `.env` file
❌ Don't share your API key
